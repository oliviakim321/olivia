THE CATASORT

My first programming language was C++, which we used at Waterloo for several programming courses. In my first year, my groupmates and I created "The Catasort" as an end of term project. The Catasort is a candy sorting robot that groups assorted candies by catapulting them into different bins according to their colour. The code for this project was written in Lego Mindstorm's RobotC and the catapult was constructed from lego, spare wood and plastic we found at our school's shop. This was a really fun project that incorporated both mechanical and software design.
